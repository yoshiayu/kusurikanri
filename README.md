# 薬管理アプリのポートフォリオ  
# (Of the drug management portfolio)

### ポートフォリオについて(About the portfolio)

このリポジトリは@yoshiayuのポートフォリオを公開するためのものです。  
This repository is for publishing @ yoshiayu's portfolio.
